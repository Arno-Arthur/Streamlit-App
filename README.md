# Разработка веб-приложения для инференса моделей машинного обучения

## Описание проекта
Данный проект представляет собой расчетно-графическую работу по дисциплине **«Машинное обучение и анализ данных»**. Основная цель — создание интерактивного веб-дашборда для вывода прогнозов (инференса) моделей машинного обучения и проведения визуального анализа данных в задаче обнаружения мошеннических транзакций (*Credit Card Fraud Detection*).

Приложение предоставляет пользователю следующие возможности:
*   Загрузка и предварительный просмотр данных.
*   Проведение разведочного анализа данных (EDA) с интерактивными графиками.
*   Визуализация зависимостей и корреляций между признаками.
*   Получение предсказаний в реальном времени от шести различных алгоритмов классификации.

## Стек технологий

| Категория | Технологии и библиотеки |

## Структура приложения
Веб-приложение построено по модульной архитектуре и состоит из четырех страниц:

### 1. О разработчике
Страница содержит информацию об авторе проекта: ФИО, номер учебной группы, университет, тему работы и фотографию.

### 2. О датасете
Подробное описание предметной области и используемых данных:
*   **Источник:** Платформа Kaggle (набор данных *Credit Card Fraud*, 1 млн записей).
*   **Признаки:** Описание 8 переменных (географические метрики, финансовые показатели, способ оплаты).
*   **Предобработка:** Информация о работе с дисбалансом классов, отсутствием пропусков и обработкой выбросов.

### 3. Анализ данных (EDA)
Страница с результатами разведочного анализа, включающая:
*   Распределение целевой переменной (соотношение классов).
*   Гистограммы и boxplot'ы для числовых признаков.
*   Тепловая карта корреляций (Pearson/Spearman).
*   Анализ влияния бинарных факторов на вероятность мошенничества.

### 4. Инференс моделей
Центральный модуль для получения прогнозов. Поддерживает два режима работы:
1.  **Ручной ввод:** Интерактивная форма для ввода параметров одной транзакции (расстояние, сумма, тип оплаты) с мгновенным выводом результата.
2.  **Массовая обработка:** Загрузка файла формата `.csv` для пакетного прогнозирования с возможностью скачивания результатов.

Пользователь может динамически выбирать одну из шести обученных моделей для выполнения предсказания.

## Используемые модели
В приложение интегрированы алгоритмы, отобранные по метрике **F1-score**:

| № | Тип модели | Алгоритм | Описание |
| :--- | :--- | :--- | :--- |
| 1 | Классическая | `DecisionTreeClassifier` | Дерево решений |
| 2 | Бустинг | `GradientBoostingClassifier` | Градиентный бустинг (Sklearn) |
| 3 | Продвинутый бустинг | `CatBoostClassifier` | Категориальный бустинг (Yandex) |
| 4 | Бэггинг | `RandomForestClassifier` | Случайный лес |
| 5 | Ансамбль | `StackingClassifier` | Стекинг мета-обучения |
| 6 | Нейросеть | `Sequential (DNN)` | Глубокая полносвязная сеть (TensorFlow) |
